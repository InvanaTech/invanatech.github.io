Deploy Invana Platform on your local machine or cloud using docker compose.

Deploy Invana Infrastructure

git clone git@github.com:invanalabs/docker-templates.git
cd docker-templates/invana-with-janusgraph
docker-compose up

This will start the services Invana Engine and Invana Studio at 8200 and 8888 ports respectively.

Setup Invana Studio

Access Invana Studio at http://localhost:8888, connect to Invana Engine server URL which defaults to http://localhost:8200/graphql.
