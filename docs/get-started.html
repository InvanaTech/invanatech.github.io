Invana is a complete Knowledge Graph solution for research or business of any size. Model the data with in your system, import your data sets, databases and any other sources of information. Link the data entities in the system with relationships and meta data to assign weights or properties to the links.

How it works

It works on top of graph databases that support Apache TinkerPop. Apache TinkerPop™ is an opensource graph computing framework for both graph databases (OLTP) and graph analytic systems (OLAP) maintained by Apache Software Foundation.

Invana is made of two core components - invana engine and invana studio, which are intentionally separated as projects to decouple the GraphQL API and the User Interface people interact with respectively.

Invana needs the following three services to serve the purpose ie., "building and exploring knowledge graphs"

  1. Graph Database: Any Apache TinkerPop enabled graph database.
  2. Invana Engine : GraphQL and Python API for Knowledge Graphs.
  3. Invana Studio: User Interface for visualising and managing data on Knowledge Graphs.
