Get Started

Get started with building and exploring first knowledge graph using Invana.

1. Starting JanusGraph server
docker run -it -p 8182:8182 janusgraph/janusgraph

This will start JanusGraph instance and the gremlin server is available at http://localhost:8182/gremlin. More installation instructions can be found here

2. Starting Invana Engine
docker run -p 8000:8000 -d -e GREMLIN_SERVER_URL=ws://xx.xx.xx.xx:8182/gremlin --name invana-engine invanalabs/invana-engine

This will start invana engine GraphQL API at 8000 port. More installation methods can be found here

3.Starting Graph Explorer
docker run -p 8888:8888 -d --name graph-explorer invanalabs/graph-explorer

This will start graph explorer on 8888 port, you can access the application at http://localhost:8888.

Currently, only JanusGraph is supported with Invana Engine.

Go to Help for more tutorials.
