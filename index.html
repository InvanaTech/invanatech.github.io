Opensource Knowledge Platform

We help people explore, analyse and find insights from data using the power of Knowledge Graphs.

Get Started

Build Knowledge Graphs

Import data from all of your information sources, be it any SQL or NoSQL databases or even CSV files. Create unifying Model and build knowledge graphs.

Invana Engine

GraphQL API for building and querying knowledge graphs.

Features include:

  • Works using Apache TinkerPop™ graph computing framework.
  • Python and GraphQL API.
  • Currently supports JanusGraph only, support for more graphs databases shall be added soon.

Explore Knowledge Graphs

Extend your curiosity, browsing through every connected data visually as force directed graphs. Utilise the full potential of graph computing queries.

Graph Explorer

Visualise and traverse through knowledge graphs.

Features include:

  • Query with Apache TinkerPop's Gremlin
  • Visualise data as force-directed Graphs
  • Visualise data as tables
  • Traverse through data extending in and out data.
  • Save your graph as Image
